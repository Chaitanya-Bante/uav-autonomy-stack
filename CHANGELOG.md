# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2024-11-09

### Added
- Waypoint navigator node with YAML mission files
- Launch files for square flight and waypoint navigation
- Multiple example missions (figure-8, climb, patrol)
- Parameter-based configuration
- Connection diagnostic tool

### Fixed
- Topic naming for PX4 v1.14+ (_v1 suffix)
- ROS_DOMAIN_ID configuration
- Home position tracking

## [0.1.0] - 2024-11-09

### Added
- Initial workspace setup
- Square flight pattern node
- Basic offboard control
- PX4-ROS2 bridge configuration
